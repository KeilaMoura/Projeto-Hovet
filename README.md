# Sistema de Agendamento para Hospital Veterinário

Este projeto refere-se ao front-end de um sistema de agendamento para um hospital veterinário, utilizando React. Atualmente, a tela de login está pronta e a tela inicial está em desenvolvimento.

## Funcionalidades

- **Tela de Login:** Permite que os usuários acessem o sistema de agendamento com suas credenciais.
- **Tela Inicial (Home):** Em produção. Exibirá próximas consultas, informações sobre os pacientes e permitirá acesso rápido a outras funcionalidades.

## Tecnologias Utilizadas

- **Frontend:** React.js
